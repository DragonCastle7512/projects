# 프로젝트 기록 포트폴리오 (React + Vite)

진행한 프로젝트들을 타임라인 형태로 정리한 포트폴리오입니다.  
프로젝트별로 **개요/아키텍처, 기술 스택, 주요 업무, 트러블슈팅, 링크**를 한 화면에서 확인할 수 있습니다.

## 주요 특징

- 연도 기준 타임라인 네비게이션
- 프로젝트 상세 패널 제공
  - 개요 & 아키텍처
  - 기술 스택
  - 주요 업무(다중 태그 지원)
  - 트러블슈팅
  - 관련 링크
- 반응형 UI
- Vite 기반 빌드/배포 구성
